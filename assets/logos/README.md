# Company logos (Work Experience)

Run `bash vendor-logos.sh` from the repo root to download these.
Filenames must match the src in index.html:
  fulcrum.png, healthnz.png, anz.png, mable.png, posmusic.png, brandkit.png, digitaldialogue.png

Any logo that is missing falls back to the company initials automatically.
To override a logo, drop a square PNG at assets/logos/<name>.png.
